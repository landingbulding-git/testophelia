# Icons Directory

This directory should contain the extension icons in the following sizes:

- icon16.png - 16x16 pixels
- icon32.png - 32x32 pixels  
- icon48.png - 48x48 pixels
- icon128.png - 128x128 pixels

## Icon Design Guidelines

- **Primary Color**: Orange (#ff6b35) to match the sphere
- **Style**: Simple, modern, recognizable
- **Format**: PNG with transparency
- **Design**: Could be a simplified orange sphere or abstract design

## Temporary Setup

For development and testing, you can:
1. Use any placeholder PNG images
2. Create simple orange circles in an image editor
3. Download free icons from icon libraries
4. Remove icon references from manifest.json if not needed

The extension will function without icons, but they're recommended for a professional appearance.
